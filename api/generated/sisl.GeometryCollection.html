applymap(func, **kwargs)

Apply a function to all elementwise

Applies the function func to each of the contained objects and returns a new collection with the applied function to them.

Parameters:

  • func (callable) – the function to be called on each object contained

  • **kwargs (optional) – keyword arguments passed directly to func

Returns:

Collection

Return type:
